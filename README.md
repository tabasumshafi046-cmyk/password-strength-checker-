# Password Strength Checker 
This project is a cybersecurity application that helps users understand the importance of strong passwords.it evaluates password strength in realtime based on common security rules used to prevent weak  and easily guessable passwords.
## Cybersecurity objective
weak passwords are one of the main reasons for cybersecurity breaches.This project demonstrates how password policies can improveaccount security by enforcing strong password practices.
## features
- minimum password length validation
- uppercase and lowercase letter checks
- number validation
- special character detection
- real-time password strength 
feedback(weak/medium/strong)
## Technologies used
- HTML
- CSS
- JavaScript
## How to Run
1. Download or clone the repository
2. open index. html in any modern web browser
3. enter a password to check its strength 
## Security Concepts covered 
- password policy enforcement
- Client-side input validation
- Basic password hygiene awareness
- Prevention of weak-password vulnerabilities
